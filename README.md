# Artificial Intelligence for Programming

jni4net is a bridge between Java and .NET. Java.Interop provides open-source bindings of Java's Java Native Interface (JNI) for use with .NET managed languages such as C#. Strange is a Quantum Computing API and simulator for Java. Erupt is a low-code, full-stack class framework. The azure-quantum python package submits jobs to the Azure Quantum service. MCP is a protocol for seamless integration between LLM applications and external data sources. The Agent2Agent (A2A) is an open protocol enabling communication and interoperability between opaque agentic applications. DeepEval is a simple-to-use LLM evaluation framework, for evaluating and testing large-language model systems. ReforgeAI is an agentic AI system designed to modernize legacy Java codebases. Copilot Chat extension for VS Code. A general-purpose Claude Code action for GitHub PRs and issues. Warp is the agentic development environment, built for coding with multiple AI agents. OpenHands platform for cloud coding agents. Protégé framework for building intelligent systems. VibeCoder is an autonomous coding agent. Kodezi Chronos Debugging-first language model. inspectIT Ocelot - Java agent. LangGraph is a orchestration framework. Cursor is The AI Code Editor. Quarkus is a Cloud Native, Container First framework. m2cgen is a library which provides an easy way to transpile trained statistical models. OpenJML is a program verification tool. Zeppelin is a web-based notebook. Cadence is a distributed orchestration engine. NVIDIA FLARE is a domain-agnostic SDK that allows data scientists to adapt existing ML/DL workflows to a federated paradigm. Manifold is a Java compiler plugin. Rascal meta-programming language. Unanimo Design System is a solution oriented to mission-critical business applications generated by GeneXus. OpenXava is an alternative development framework. Supabase gives you a dedicated Postgres database to build your web, mobile and AI applications

The directory structure includes a specific folder for each phase of development and engineering programs:

- __MBSE__: The root folder Arcadia (ARChitecture Analysis and Design Integrated Approach) is a method of tools dedicated to systems engineering and architecture, supported by the Capella modeling tool.    
  - __CAD__: For computer-aided design.
  - __CAM__: Dedicated to the part manufacturing and assembly process.
  - __CAS__: For simulation of end-to-end functionality and performance analysis.
