# Legacy to Modern

Zeppelin is a web-based notebook that enables data-driven, interactive data analytics and collaborative documents with SQL, Scala and Apache Spark. Spark RAPIDS plugin - accelerate Apache Spark with GPUs. AiiDA is a workflow manager for computational science with a strong focus on provenance, performance and extensibility. NVIDIA FLARE (NVIDIA Federated Learning Application Runtime Environment) is a domain-agnostic, open-source, extensible Python SDK that allows researchers and data scientists to adapt existing ML/DL workflows to a federated paradigm. Open Interpreter lets LLMs run code (Python, Javascript, Shell, and more) locally. MARCO is a prototype compiler for the Modelica language, and is based on the LLVM / MLIR compiler technology. Sulong is a high-performance LLVM bitcode runtime built on the GraalVM. Rascal meta-programming language. LangChain4j is a Java library that simplifies the integration of LLMs into Java applications. A unified library of SOTA model optimization techniques. ZODB provides an object-oriented database for Python that provides a high-degree of transparency. OpenXava is a framework for developing effective management applications. n8n is a workflow automation platform. Stone Framework is a framework for GeneXus that offers a range of functionalities and data patterns to enhance productivity and build robust, scalable applications. Unanimo Design System is a solution oriented to mission-critical business applications and the back-office generated by GeneXus. OpenSourceCOBOL4J is a COBOL compiler that translates COBOL programs to Java programs. Airbyte is a leading data integration platform that offers ETL. jOOQ is a library that generates Java code from your database metadata. SQLines is a set of tools designed to assist with database migration.
